## 1. Design System

### 1.1 Color Palette

| Token | Value | Usage |
|-------|-------|-------|
| `primary.900` | Deep Teal | App background, gradients |
| `primary.700` | Teal | Secondary backgrounds |
| `primary.600` | Medium Teal | Card backgrounds |
| `primary.400` | Light Teal | Borders, dividers |
| `accent.500` | `#EFC2B3` (Soft Peach) | Primary CTAs, highlights |
| `accent.400` | Lighter Peach | Text highlights, icons |
| `accent.600` | Darker Peach | Hover states |

### 1.2 Typography

- **Primary Font**: Geist Sans
- **Monospace**: Geist Mono
- **Scale**: Tailwind default (sm, base, lg, xl, 2xl, 3xl, 4xl)

### 1.3 Spacing & Layout

- **Base Unit**: 4px
- **Touch Targets**: Minimum 44Ã—44px
- **Container Max Width**: `max-w-7xl` (1280px)
- **Card Radius**: `rounded-xl` (12px), `rounded-2xl` (16px)
- **Button Radius**: `rounded-xl` (12px), `rounded-full` (pills)

### 1.4 Elevation & Effects

- **Blur**: `backdrop-blur-lg` for overlays
- **Shadows**: `shadow-lg`, `shadow-2xl` for modals
- **Gradients**: `gradient-to-br`, `gradient-to-r`

## 9. Interaction Patterns

### 9.1 Touch Targets

- **Minimum Size**: 44Ã—44px
- **Button Heights**: 48px (small), 52px (medium), 56px (large)
- **Icon Buttons**: Min 44Ã—44px with 10px padding

### 9.2 Button States

| State | Visual Treatment |
|-------|-----------------|
| Default | Background color, no shadow |
| Hover (Desktop) | Lighter background, subtle shadow |
| Active/Pressed | Darker background, no shadow |
| Focus | 2px ring offset, accent color |
| Disabled | 50% opacity, cursor not-allowed |
| Loading | Spinner + disabled state |

### 9.3 Form Inputs

- **Height**: 48-52px minimum
- **Border**: 1px solid primary.400
- **Focus**: Border color changes to accent.400
- **Error**: Red border, error message below
- **Labels**: Above input, medium weight

### 9.4 Cards

- **Padding**: 16-24px (mobile) / 20-32px (desktop)
- **Border Radius**: 12px (rounded-xl) or 16px (rounded-2xl)
- **Border**: 1px solid with opacity
- **Hover**: Subtle background change, box-shadow

### 9.5 Checkboxes

- **Style**: Custom circular checkboxes
- **Size**: 28-32px diameter
- **Completed**: Filled accent color with white checkmark
- **Uncompleted**: Border only, transparent fill

## 11. Animation Specifications

### 11.1 Page Transitions

- **Duration**: 300-400ms
- **Easing**: Spring (damping: 25, stiffness: 300)
- **Pattern**: Fade + slide (y: 20px)

### 11.2 Component Animations

| Component | Animation | Duration |
|-----------|-----------|----------|
| Cards appearing | Fade + slide up | 300ms + stagger 50ms |
| Modal open | Slide from bottom (mobile), scale up (desktop) | 300ms spring |
| Modal close | Reverse of open | 200ms |
| Progress ring | Scale from 0 + stroke dash animation | 500ms |
| Accordion expand | Height auto + opacity | 300ms |
| Checkbox check | Scale pulse | 200ms |
| Tab switch | Content crossfade | 300ms |

### 11.3 Micro-interactions

- **Button tap**: Scale down to 0.98, then back
- **Card hover**: Subtle lift (shadow-lg)
- **Toggle**: Spring bounce
- **Timer countdown**: Pulse on completion

### 11.4 Future: Reduced Motion

Support `prefers-reduced-motion`:
- Disable all transform animations
- Keep opacity fades
- Use `initial={false}` after first load
